## Youtube Video Summarizer
THis is youtube video summarizer web with react framework and python as backend 


## To setup it :
<h6>
Backend
</h6>

```
cd server 
Go to .env add the gemini api url 
python -m venv env 
pip install -r requirement.txt 
```



<h6>
Frontend
</h6>

```
cd client
npm install 
npm start
```
